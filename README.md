# Snottyburger 2.0

Base Next.js app for a burger shop site with three surfaces prepared: `/home`, `/orden`, and `/panel`.

## Available Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`

## Current Scope

- Production-style `/orden` ordering flow with typed mock data
- Mobile-first catalog, product customization modal, location selector, and cart drawer
- Shared domain structure prepared for future backend integration
